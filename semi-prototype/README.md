# Semi Working Prototype

I wrapped all our work from last week into a semi-working prototype.

Not sure if we want to use this or not..?

## To Use:

1. Create python virtual env
2. Activate env
3. Download requirements through pip
4. Create .env file and add Keys to it like so:
```
GEMINI_API_KEY=
MISTRAL_API_KEY=
HF_TOKEN=
```
5. Run it through the following: `python -m prototype.main`
