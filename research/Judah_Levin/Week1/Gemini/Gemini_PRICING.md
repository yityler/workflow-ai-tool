# Gemini Pricing for Agent Usage

Source: [Gemini Developer API pricing](https://ai.google.dev/gemini-api/docs/pricing), last updated June 9, 2026.

Google states that agent usage is charged based on the underlying model's token consumption and tool usage. The table below uses the Standard paid-tier list prices for 1 million text input tokens and 1 million text output tokens. Tool costs, context caching, batch, flex, priority inference, and media-specific output pricing are not included.
