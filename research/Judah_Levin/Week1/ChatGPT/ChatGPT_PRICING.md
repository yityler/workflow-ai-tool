# ChatGPT Pricing Per Million Tokens for AI Agents

Source: [OpenAI API pricing](https://developers.openai.com/api/docs/pricing), checked June 14, 2026.

These are Standard, short-context prices per 1 million tokens when building or running an AI agent with OpenAI models. OpenAI states that Responses API, Chat Completions API, Realtime API, Batch API, and Assistants API are not priced separately; tokens are billed at the selected model's input and output rates. Batch, Flex, Priority, long-context, cached-input, regional processing, built-in tool, file storage, and fine-tuning prices may differ.
