# Pricing & Usage Limits

> Prices and limits change often — these are approximate as of mid-2026. Costs are per
> **1 million (1M) tokens** (roughly 750,000 words); input and output are charged separately.

### Mistral AI (Mistral Small)
| Pricing | Usage Limits |
|---------|--------------|
| Free experimental plan available ($0 to test) | Rate-limited (capped requests per second) |
| ~$0.10–0.30 per 1M input tokens (paid) | Monthly token allowance on the free tier |
| ~$0.30–0.60 per 1M output tokens (paid) | Enough for testing and small projects |
| Get a key: https://console.mistral.ai/api-keys | |

### Reka AI (Reka Flash)
| Pricing | Usage Limits |
|---------|--------------|
| Free trial credits on sign-up (no permanent free plan) | Limited by the trial credit balance |
| ~$0.20–0.40 per 1M input tokens (paid) | Standard rate limits apply |
| ~$0.40–0.80 per 1M output tokens (Reka Core costs more) | |
| Get a key: https://platform.reka.ai/apikeys | |

### Cohere (Command R)
| Pricing | Usage Limits |
|---------|--------------|
| Free trial API keys for evaluation ($0 to test) | ~20 calls per minute (trial) |
| ~$0.15 per 1M input tokens (paid) | ~1,000 calls per month (trial) |
| ~$0.60 per 1M output tokens (Command R+ costs more) | Production keys remove these caps |
| Get a key: https://dashboard.cohere.com/api-keys | |

## Summary
| Provider | Free option | Free-tier limits | Rough paid cost (per 1M tokens) |
|----------|-------------|------------------|----------------------------------|
| Mistral  | Free plan | Rate + monthly token cap | ~$0.10–0.30 in / ~$0.30–0.60 out |
| Reka     | Trial credits | Limited by credit balance | ~$0.20–0.40 in / ~$0.40–0.80 out |
| Cohere   | Trial key | ~20/min, ~1,000/month | ~$0.15 in / ~$0.60 out |
