# Cornerstone-Tabs-Configurator-Metafields

## Background
- Cornerstone-Tabs-Configurator-Metafields (started/repo owned by roel-c) is a template system for makeing PDPs (Product Detail Pages)
- link to repo page [here](https://github.com/roel-c/cornerstone-tabs-configurator-metafields)

## Pros

- Incredibly wide range of possibilities

- large numbers of styles

- has built in button/link system

## Cons

- No integrated AI editing system (yet)

- Very complex

- large number of files (makes you dig for what you want)
